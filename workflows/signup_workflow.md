"# 회원가입 자동화 Workflow

## 목표
사용자가 회원가입 폼을 제출하면, 자동으로 계정을 생성하고 로그인 상태로 전환합니다.

## 입력값
- `username`: 사용자 이름 (문자열)
- `password1`: 비밀번호 (문자열)
- `password2`: 비밀번호 확인 (문자열)

## 처리 단계
1. 입력값 유효성 검사
2. 사용자 계정 생성
3. 로그인 상태 전환
4. 홈 페이지로 리디렉션

## 예외 처리
- 비밀번호 불일치: 에러 메시지 표시
- 중복된 사용자 이름: 에러 메시지 표시
- 약한 비밀번호: 에러 메시지 표시

## 결과
- 성공: 홈 페이지로 리디렉션
- 실패: 에러 메시지 표시 및 회원가입 폼 재표시
"
